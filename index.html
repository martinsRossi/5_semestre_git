Comandos Git

Comando Descrição
git clone Copiar o repositório do github para nossa
git branch Mostra as branchs locais
git checkout -b feat/gitBasic Cria uma branch chamada "feat/gitBrasic"

Sequência de Comandos - PUSH

1. Primeiro, adicionaremos os arquivos criados/modificados

git add . (ou arquivo)
O "." significa todos os arquivos. Caso queira algum arquivo, escreva ele. Por exemplo: git add README.md

2. Caso seja a primeira vez que fará um commit nesta máquina, precisará fazer os seguintes comandos:

Clique aqui

3. Uma vez configurado, executamos:

git commit -m "seu_comentário"
O comentário é muito importante para prover o máximo de informações sobre o commit feito

4. Para verificar o status, rodamos:

git status

5. Por fim, executamos

git push -u origin nome/branch
